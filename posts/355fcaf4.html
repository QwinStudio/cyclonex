手把手教你打造一个属于自己的博客 GitHub+Hexo

Gebit Qwin2023-09-302023-12-16

准备工作

1.首先你要下载 Node.js
官网: https://nodejs.org/en
选择18.18.0LTS下载就可以了 博客用不了什么新功能
2.下载Git
官网: https://nodejs.org/en
选择Download for Windows 并安装 尽量安装到系统盘以外

配置GitHub存储库

*由于GitHub DNS问题 懂得都懂 要看网速和运气的 如果都没有 请看下一章Gitee
1.注册GitHub账号(有可以直接忽略)
2.登录后点击右上方你的头像 找到”Your Repositories”(你的存储库) 点进去
3.点击绿色的”New”选项
4.在Owner的Repository name下填上 你的GitHub账号名字+github.io (如: qwinstudio.github.io)
5.在Initialize this repository with选上Add a README file(这一步很重要!!)
6.点击 Create repository 完成添加
7.再点进去你的存储库
恭喜你完成了存储库的添加!

配置SSH密钥

1.打开运行(菜单键+R)
2.输入cmd回车
3.输入 : ssh-keygen -t rsa -b 4096 -C “你的电子邮箱(要和GitHub的注册邮箱一样)”
4.打开资源管理器(菜单键+E)
5.进入系统盘(默认为C)→用户→你的用户名或Admin→.ssh
6.找到id_rsa.pub文件用记事本打开并复制其中的内容
7.打开GitHub 点击右上方头像 打开Settings选项
8.点击Settings左边[Access]下的SSH and GPG keys
9.右侧点击绿色的”New SSH Key”
10.在新页面中将Title填上任意名称 在Key下面填上刚刚复制的SSH Key
11.点击绿色的Add SSH Key完成SSH密钥添加
